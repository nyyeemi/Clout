# Clout

A mobile photo competition application where users submit photos, vote anonymously on image pairs, and view results through recurring competitions.

---

## Project Focus

The primary goal of this project is learning through implementation. Key areas of focus include:

  - Building a mobile application end to end
  - Applying best practices in API design and data modeling
  - Structuring a maintainable frontend and backend codebase
  - Working with containerized development environments
  - Collaborating in a small team using version control and pair programming

## Current Key Features
  - Anonymous photo submissions and voting
  - Pairwise image comparisons with non-repeating pairs per user
  - Ranking based on aggregated pairwise results
  - Leaderboards for completed competitions
  - User profiles with past entries and follower relationships
  - Feed for viewing activity from followed users
  - Basic web-based admin interface
